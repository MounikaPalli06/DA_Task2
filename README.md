# Exploratory Data Analysis (EDA) & Business Intelligence Project

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) and Business Intelligence analysis using Python, SQL, and Google Colab. The main objective is to uncover patterns, trends, and relationships within the dataset and generate meaningful business insights.

---
# Objectives
- Perform data cleaning and preprocessing
- Generate descriptive statistics
- Create data visualizations
- Analyze relationships between variables
- Write SQL queries for business questions
- Develop KPI metrics
- Create dashboard mockups

# Dataset Information
Dataset Name:

Retail Store Sales Dataset

The dataset contains:
- Order details
- Product information
- Sales data
- Profit data
- Marketing spend
- Customer information
- Region-wise sales

---

# Dataset Columns

| Column Name | Description |
|-------------|-------------|
| Order_ID | Unique order identifier |
| Customer_Name | Customer full name |
| Product | Product purchased |
| Category | Product category |
| Region | Sales region |
| Quantity | Quantity purchased |
| Sales | Total sales amount |
| Profit | Profit earned |
| Marketing_Spend | Marketing expenditure |
| Payment_Method | Payment type |
| Order_Date | Date of order |
